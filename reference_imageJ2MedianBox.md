## imageJ2MedianBox
<img src="images/mini_empty_logo.png"/><img src="images/mini_empty_logo.png"/><img src="images/mini_clijx_logo.png"/><img src="images/mini_empty_logo.png"/>

Apply ImageJ2 / ImageJ Ops Median filter with a Sphere shape to an image.

Categories: Denoise, [Filter](https://clij.github.io/clij2-docs/reference__filter)

Availability: Available in Fiji by activating the update sites clij, clij2 and clijx-assistant-extensions.
This function is part of clijx-assistant-imagej2_-0.4.1.0.jar.

### Usage in ImageJ macro
```
Ext.CLIJx_imageJ2MedianBox(Image source, Image destination, Number radius_x, Number radius_y, Number radius_z);
```
